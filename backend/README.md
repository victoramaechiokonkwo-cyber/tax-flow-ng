# Tax-Flow NG Backend API

Data Reporting and Taxation System backend for legacy institutions in Nigeria.

## Features

- **Authentication**: JWT-based auth with role-based access control
- **Tax Calculations**: CIT, VAT, PAYE, WHT with Nigerian tax laws
- **Transaction Management**: CRUD operations for tax filings
- **Report Generation**: FIRS-compliant Excel/PDF reports
- **Data Extraction**: ETL pipeline for legacy database integration
- **Audit Logging**: Complete activity tracking
